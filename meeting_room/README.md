# 简易会议室预约系统

## 原始开发需求

实现一个简单会议室预定系统。分步递进：

1. 基础功能：实现预定、取消功能
2. 基础数据结构、冲突检查算法
3. 系统需要支持多个用户同时预定
4. 需要正确处理多用户同时预定同一时间段
5. 需要考虑死锁的防止
6. 各团队有一些不同的些业务规则，例如：
   - 单次预定不超过4小时
   - 同一用户每天最多3个时段
   - 不能预定过去时间
   - 必须提前15分钟预定
   - 过期30分钟自动释放

## 运行说明

最小可运行版本采用 Python 标准库实现，无需安装第三方依赖。

启动命令：

```bash
python3 -m app.server
```

访问地址：

```text
http://127.0.0.1:8000
```
