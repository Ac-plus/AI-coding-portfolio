**问题描述**

高松灯喜欢收集石头。某天她想把石头送给千早爱音，但书包容量不足，于是只能认真选取几个。设她共有 `n` 个石头，它们的重量和价值分别为 `weight[i]` 和 `price[i]`（`i = 0, 1, ..., n-1`）；书包的容量为 `capacity`。高松灯应该选取哪些石头装进书包才能使总价值最大？需要以 `n` 元组的形式给出最优装法，同时求出在此最优解下的最大价值。

**输入**

- 高松灯的石头总数 `n` ，书包容量 `capacity`
- `n` 个石头的重量数组 `weight` 和价值数组 `price`

**输出**

- `n` 个元素组成的 bool 数组，表示每个石头是否装入书包

**示例1**

- 输入：`n = 3`, `capacity = 10`, `weight = [4,6,1]`, `price = [20,50,15]`
- 输出：`[1,1,0]`
